Open Source

PDF2Text Pilot 2.0 is an open source product that uses commercial the PDF library component. So, the program code, excepting code of the PDF Creator Pilot library, is available for free.

You can download PDF2Text Pilot source code as ZIP archives here: http://www.colorpilot.com/pdf2text.html#source

 

See more details about the PDF Creator Pilot library... http://www.colorpilot.com/pdflibrary.html
