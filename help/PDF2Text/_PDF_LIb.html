PDF Library

PDF2Text Pilot uses the PDF Creator Pilot library for extracting text from PDF documents.

Also PDF Creator Pilot allows:

  • Using high-quality fonts to show text in different languages
  • Drawing images in popular formats
  • Creating protected PDF documents
  • Reading, merging and modifying PDF documents

PDF Creator Pilot library supports .NET, VB, ASP, Delphi, Visual C++, VBScript...

 

More details at: http://www.colorpilot.com/pdflibrary.html
